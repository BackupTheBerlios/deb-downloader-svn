 INTRODUCCIÓ
~~~~~~~~~~~~~

	Aquesta és una utilitat escrita per mi anomenada deb-downloader, una eina implementada per a una distribució
de GNU/Linux especifica anomenada Debian (o distribucions basades en ella). deb-downloader va ser escrit fa dos mesos
(06-08-2004) quan la meva connexió "d'alta velocitat" va ser tallada per un canvi de domicili. Això va fer-me pensar 
en quin seria la millor manera per actualitzar la meva Sid (Debian té tres versions (estable, testeix i inestable, 
també anomenada Sid, una llegenda urbana diu que prové de "System In Development")).

	Havia sentit parlar sobre una eina anomenada apt-zip, pero no tinc una distribucio GNU/Linux a la feina, per tant 
em vaig decidir a escriure aquesta eina. deb-downloader està escrit en Perl, per tant, l'únic requeriment per usar-lo es
disposar d'un sistema operatiu amb un interpret de perl com poden ser sistemes *nix, Macintosh (OS 7-9 i X), VMS, Windows, 
etc. (mira http://www.cpan.org/ports/index.html per més informació).

	Espero que sigui útil per tothom que la faci servir.


 REQUERIMENTS
~~~~~~~~~~~~~~

	Només hi ha dos requeriment per usar deb-downloader.
	
		- Debian GNU/Linux (o basada en Debian) amb interpret de perl per deb-downloader-build i deb-downloader-install.
		- Interpret de perl amb els mòduls Net::FTP i Net::HTTP per deb-downloader-download.


 INSTAL·LACIÓ
~~~~~~~~~~~~~~

	- tar xvfz deb-downloader-nn.nn.nn.tar.gz target_dir.
	- tar xvfj deb-downloader-nn.nn.nn.tar.bz target_dir.

 ÚS
~~~~

	Aquest script esta dividit físicament en tres parts. La primera i la tercera s'executen en una distribució Debian i
la segona en una màquina amb qualsevol sistema operatiu que tingui disponible un interpret de perl.

	Aquests són els tres scripts :

	- deb-downloader-build : Aquesta opció s'úsa per recol·lectar les uris de tots els paquets .deb que posteriorment
	  seran descarregats. Aquesta opció s'executa en una distribució Debian.

		deb-downloader-build --file=filename install package1...packagen|dist-upgrade [-d|--debug] [--skip-update]
		
			-d|--debug : Serveix per veure missatges extra de l'execució. Important per reportar bugs.
			--file : Nom de l'arxiu on gravarem les uris dels .deb.
			--skip-update : No executar apt-get update (amb aquesta opció activada, no és necessari executar
			  el script com a usuari root.
			install paquet1...paquetn : Seleccionar els paquets a descarregar (apt-get install elquesigui).
			dist-upgrade : Actualitza la distro (apt-get dist-upgrade).			  
		

	- deb-downloader-download : Aquesta opció és usada per descarregar els paquets .deb necessaris. Pot ser executat a qualsevol
	    		ordinador amb qualsevol sistema operatiu amb un interpret de perl.
				 
		deb-downloader-download --file=filename1...filenamen [-d|--debug] [--dd-root=directory] [--skip-downloaded]
		
			-d|--debug : Serveix per veure missatges extra de l'execució. Important per reportar bugs.
			--file : Noms dels arxius d'on llegirem les uris.		
			--dd-root : Especifica el directori de destí per descarregar els paquets .deb (directori dd-root per defecte).
			--skip-downloaded : Si aquesta opció està activada, els paquets ja descarregats no es descarregaran una
			  altra vegada (si has parat l'execució abans de que s'acabi).
				 
	- deb-downloader-install : Aquesta opció és usada per instal·lar els paquets dins de /var/cache/apt/archives i, si es vol, 
		    	executar apt-get dist-upgrade, apt-get install el que sigui o res, com vulguis.
		
		deb-downloader-install --file=filename [-d|--debug] [install package1...packagen|dist-upgrade][--mirror-format] [--dd-root=directory]
		
			-d|--debug : Serveix per veure missatges extra de l'execució. Important per reportar bugs.
			--file : Nom dels arxius d'on llegirem les uris.		
			--dd-root : Especifica el directori d'on agafarem els paquets .deb (directori dd-root per defecte).
			--mirror-format : Aquesta opció ens gravarà els paquets amb el format de directoris que té el servidor 
			  (incompatible amb la funcio dd-root no informada).		
			install paquet1...paquetn : Seleccionar els paquets a descarregar (apt-get install elquesigui).
			dist-upgrade : Actualitza la distro (apt-get dist-upgrade).			  
			copy-files : Només copia els arxius al directori especificat.
